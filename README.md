# e-plantShopping

e-plantShopping is a React-based shopping cart application for an online plant shop.  
It allows users to browse different house plants, view details, and add items to a shopping cart for purchase.

## Features

This application includes:

- A home (landing) page with a button that takes users to the product list  
- A navigation bar to move between home, products, and the cart  
- Individual plant cards showing image, name, description, price, and an option to add to cart  
- Multiple plant categories (at least three), such as aromatic or medicinal plants  
- A cart page where selected items are displayed  
- For each item in the cart: image, price per unit, total price, controls to increase/decrease quantity, and a remove option  
- Buttons to continue shopping or proceed to checkout
